# 📈 Student Grade Tracker & Manager

An interactive console program tracking student scores, reporting summary statistics, and printing letter grades.

## 🛠️ Features

- **Class Size Flexibility:** Allows inputting dynamic arrays of student details.
- **Continuous Grade Input:** Captures multiple test/subject marks for each student.
- **Aggregate Calculations:** Computes average scores, tracks highest/lowest marks, and maps class performance to standardized letter grades (A, B, C, etc.).

---

## 🏗️ Structure Overview

- **`Student`**: Holds name, roll number, list of numeric grades, and wraps statistics calculation logic.
- **`Student_Grade_Manager_Tracker`**: Orchestrates console input prompts and formats the final summary reports.

---

## 🚀 How to Run

1. Open your terminal in this directory:
   ```bash
   cd "Student Grade Tracker Manager"
   ```

2. Compile all files:
   ```bash
   javac *.java
   ```

3. Run the driver class:
   ```bash
   java Student_Grade_Manager_Tracker
   ```
